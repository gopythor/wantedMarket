# wantedMarket
## 프로젝트 주제
스프링부트를 통한 원하는 물건 가격 기재하여 구매자에게 판매자가 가격을 제시하는 역경매 기능 구현   
목표 : 간이 경매용 사이트 개설하여 구매자와 판매자간 가격을 조율할 수 있는 서버 구현
